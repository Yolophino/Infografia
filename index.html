Herramientas y Servicios de Internet

Protocolos y tecnologías que permiten la comunicación y el intercambio de información en la red

HTTP

Protocolo utilizado para transferir páginas web. Permite la comunicación entre navegadores y servidores.

FTP

Protocolo para transferir archivos entre computadoras a través de una red.

SMTP

Protocolo utilizado para el envío de correos electrónicos entre servidores.

SSH

Permite conectarse de forma segura a un servidor remoto mediante cifrado.

Telnet

Protocolo para acceso remoto a equipos, pero sin cifrado, por lo que es menos seguro que SSH.

IRC

Sistema de chat en tiempo real basado en canales de comunicación en Internet.

VoIP

Tecnología que permite realizar llamadas de voz a través de Internet.

P2P

Modelo de red donde los equipos comparten recursos directamente sin un servidor central.

IPTV

Servicio que transmite contenido televisivo a través de Internet.
