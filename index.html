Tecnologías de Internet

El Ecosistema de los Motores de Búsqueda

a. ¿Qué es un Motor de Búsqueda?

Es un sistema informático diseñado para buscar información en la World Wide Web (WWW). Los resultados se presentan generalmente en una lista de resultados que pueden consistir en páginas web, imágenes, videos y otros tipos de archivos, basados en palabras clave que ingresa el usuario.

b. Índices vs. Motores de Búsqueda

Índices (Directorios)

Construidos y organizados manualmente por seres humanos en categorías y subcategorías temáticas (Ej. el antiguo Yahoo Directory).

Motores de Búsqueda

Automatizados. Utilizan programas ("arañas" o bots) para recorrer la web, recolectar datos y algoritmos para organizar los resultados.

Núcleo Tecnológico

c. ¿Cómo trabajan internamente?

1

Rastreo (Crawling)

Bots o "arañas" viajan de enlace en enlace recolectando el contenido público de las páginas web.

2

Indexación (Indexing)

El texto y archivos rastreados se organizan y almacenan en una base de datos gigante llamada Índice.

3

Clasificación (Ranking)

Algoritmos evalúan el índice y deciden qué páginas son más relevantes y útiles para la consulta del usuario.

e. Metabuscadores

Un "meta-search" no tiene una base de datos ni rastreadores propios.

¿Para qué sirve? Recibe tu consulta y la reenvía a múltiples motores de búsqueda tradicionales simultáneamente. Luego, recopila, filtra y elimina resultados duplicados para presentarte una lista unificada y más completa. (Ej. Dogpile, Trivago para hoteles).

f. Archivo robots.txt

Es un archivo de texto alojado en la raíz de un sitio web (ej. tusitio.com/robots.txt).

¿Para qué sirve? Actúa como el "policía de tráfico" del sitio. Le indica a los bots rastreadores (como el de Google) qué áreas o archivos del sitio web tienen permitido analizar y cuáles tienen prohibido acceder (para evitar sobrecarga del servidor o mantener privacidad en ciertas carpetas).

g. Búsqueda de Mapas en Internet

Servicios como Google Maps o Apple Maps funcionan combinando múltiples tecnologías complejas:

  • ✓ GIS y Satélites: Utilizan Sistemas de Información Geográfica y fotogrametría satelital para trazar la geografía física.
  • ✓ Bases de Datos Locales: Indexan millones de registros de negocios (directorios locales) atados a coordenadas GPS exactas (latitud y longitud).
  • ✓ Algoritmos de Enrutamiento: Usan la teoría de grafos (como el algoritmo de Dijkstra) para calcular la ruta más corta/rápida entre dos nodos (puntos), considerando datos de tráfico en tiempo real aportados por otros usuarios (crowdsourcing).
