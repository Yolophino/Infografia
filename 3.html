Fundamentos de Sistemas Operativos (Windows Server)

💻 Windows Server

Windows Server es un sistema operativo diseñado para administrar redes, recursos y servicios dentro de organizaciones. Permite controlar usuarios, aplicaciones, seguridad y comunicación entre múltiples computadoras.

⚙️ Instalación de Windows Server

El proceso de instalación permite preparar el sistema operativo para administrar servidores y recursos de red dentro de una organización.

Requisitos del sistema

Antes de instalar, se deben verificar recursos como procesador, memoria RAM, almacenamiento y compatibilidad del hardware.

Proceso de instalación

Incluye iniciar desde un medio de instalación, elegir versión del sistema, configurar particiones del disco y establecer las opciones básicas del servidor.

Configuración inicial

Después de instalar el sistema se asigna nombre al servidor, contraseña del administrador, dirección IP y ajustes básicos de seguridad.

Administrador del servidor

Herramienta principal para agregar roles, configurar servicios y administrar los recursos del sistema.

🌐 Servicios de infraestructura básica

Los servicios de infraestructura permiten que la red funcione de forma organizada, segura y eficiente dentro de una empresa.

Active Directory

Servicio que administra usuarios, computadoras y permisos dentro de un dominio. Permite controlar el acceso a los recursos de la red.

DNS (Domain Name System)

Traduce los nombres de dominio en direcciones IP para que los dispositivos puedan localizar servidores y servicios en la red.

DHCP

Asigna automáticamente direcciones IP a los dispositivos conectados a la red, evitando configuraciones manuales.

Servicios de archivos y almacenamiento

Permiten compartir carpetas, administrar permisos y almacenar información de forma centralizada.

Servicios de seguridad

Incluyen control de accesos, políticas de grupo, autenticación de usuarios y protección de datos.
