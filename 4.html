Fundamentos de Sistemas Operativos y Windows Server

Fundamentos de Sistemas Operativos

Un sistema operativo es el software encargado de administrar los recursos del hardware y permitir la interacción entre el usuario, los programas y los dispositivos de una computadora.

Almacenamiento de datos

Permite guardar información en dispositivos como discos duros, SSD o almacenamiento en red para que pueda ser utilizada posteriormente.

Manejo del sistema de archivos

Organiza los archivos en carpetas y estructuras jerárquicas, facilitando su acceso, modificación y administración.

Manejo de procesos

Administra los programas que se ejecutan en el sistema, controlando el uso del procesador y la memoria.

Manejo del Shell

Interfaz que permite al usuario comunicarse con el sistema operativo mediante comandos o interfaces gráficas.

Administración del sistema

Incluye la gestión de usuarios, recursos del hardware y configuración general del sistema.

Comunicaciones y redes

Permite que los dispositivos intercambien información mediante protocolos y conexiones de red.

Administración de redes

Gestiona la conexión entre computadoras, servidores y dispositivos dentro de una infraestructura de red.

Protección y seguridad

Controla accesos, protege información y previene amenazas informáticas.

Operación en la nube

Permite ejecutar aplicaciones y almacenar información en servidores remotos accesibles desde internet.

Fundamentos de Windows Server

Windows Server es un sistema operativo diseñado para administrar infraestructuras de red, servicios empresariales y recursos compartidos dentro de una organización.

Instalación de Windows Server

Consiste en preparar el servidor mediante la instalación del sistema, configuración del hardware, asignación de direcciones IP, creación de cuentas administrativas y ajustes iniciales.

Servicios de infraestructura básica

Incluyen herramientas esenciales como Active Directory, DNS, DHCP, servicios de archivos y políticas de seguridad que permiten administrar redes empresariales.
