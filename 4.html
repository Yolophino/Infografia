🖥️ Administración de Redes

Fundamentos, gestión de cuentas y mantenimiento del sistema

🌐 Fundamentos de Administración de Redes

Concepto y tipos de red

Conjunto de dispositivos interconectados (LAN, WAN, MAN).

Modelo cliente-servidor

Los clientes solicitan recursos y los servidores los proporcionan.

Direccionamiento IP

Identifica de forma única a cada dispositivo en la red.

Servicios de red

DNS, DHCP y Active Directory gestionan recursos y usuarios.

Seguridad básica

Protección mediante firewalls, contraseñas y políticas.

👤 Cuentas de Computadoras

Tipos de cuentas

Usuarios, equipos y administradores con distintos niveles de control.

Gestión de usuarios

Creación, modificación y eliminación de cuentas.

Permisos y acceso

Definen qué recursos puede usar cada usuario.

Autenticación

Verifica identidad; autorización define acciones permitidas.

Entornos de dominio

Centralizan la administración de cuentas en la red.

🛠️ Mantenimiento del Sistema Operativo de Red

Actualizaciones

Parches para mejorar seguridad y rendimiento.

Monitoreo

Supervisión del estado y desempeño del sistema.

Backups

Respaldo y recuperación ante fallos o pérdida de datos.

Recursos

Gestión de CPU, memoria y almacenamiento.

Seguridad

Uso de antivirus, firewall y registros de actividad.

🔗 Integración de los Elementos

La administración de redes integra usuarios, servidores y mantenimiento para garantizar el funcionamiento continuo. Los usuarios interactúan con el servidor mediante cuentas, las cuales son gestionadas dentro del sistema. El mantenimiento asegura que los servicios permanezcan disponibles, seguros y eficientes.

Usuario → Servidor

Acceso a recursos mediante autenticación.

Gestión de cuentas

Control centralizado de usuarios y permisos.

Mantenimiento

Garantiza disponibilidad y seguridad del sistema.

Flujo general

Usuario solicita → servidor valida → sistema responde → monitoreo continuo.
